# passos-proj-site-db
 Projecto de site com sqlite flask e html e tal

1. Criar ambiente virtual
    ```sh
        python -m venv .venv
    ```
 
2. Entrar no ambiente virtual
   * **Linux**
        ```sh
            source .venv/bin/activate
        ```
   * **Windows**
        ```sh
            .venv\Scripts\activate
        ```

3. Instalar pacotes
    ```sh
        pip install -r requirements.txt
    ```
